See http://http://web.cs.dal.ca/~skelly/csci3154-mspacman/ for detailed information about this assignment.

NOTE: My own work consists of the contents within AIControllerPacMan.java

This distribution contains the following:

mspacman-sensors.pdf -- complete description of the state variables available in Ms. Pac-Man

src/AIControllerPacMan.class -- basic Ms. Pac-Man controller class (compiles)

src/AIControllerPacMan.java -- basic Ms. Pac-Man controller class (source) MAIN LOGIC FOR PACMAN AI

src/behaviour.txt -- a brief description of the controller's behaviour

src/compile-unix.sh -- compile script for Ms. Pac-Man controller under Unix/Linux

src/compile-windows.bat -- compile script for Ms. Pac-Man controller under Windows

src/controllers.osc.jar -- super class for Ms. Pac-Man controller, and other networking dependencies

src/data -- images and data structures required by the Ms. Pac-Man simulator

src/killPacman.sh -- handy script/command to kill stray processes in Unix/Linux

src/MsPacManSimulator.jar -- Ms. Pac-Man simulator

src/run-unix.sh -- script to run simulator and controller in Unix/Linux 

src/run-windows.bat -- script to run simulator and controller in Windows
